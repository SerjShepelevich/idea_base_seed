# База идей (seed)

Заготовка общей базы идей по концепции навыков (skill-packs) и архитектуре агента.

## Как пользоваться

- Карточки: `docs/ideas/cards/`

- Индекс для поиска/фильтрации: `docs/ideas/index.json`

- Принятие в проекты: фиксировать в `docs/DECISIONS.md` и `skills.lock.yaml` проекта

## Текущий набор идей

- **IDEA-0001** — Один универсальный агент, много навыков
- **IDEA-0002** — Навык как организованная папка файлов
- **IDEA-0003** — Прозрачные скрипты вместо «чёрных ящиков»
- **IDEA-0004** — Прогрессивное раскрытие контента навыка
- **IDEA-0005** — Версионируемая экспертиза (git как память навыков)
- **IDEA-0006** — Три класса навыков: базовые, 3rd-party, корпоративные
- **IDEA-0007** — Модульная архитектура агента: loop + runtime + MCP + skills
- **IDEA-0008** — Непрерывное улучшение через сохранение процедур
- **IDEA-0009** — CPU/OS/Apps: модель как CPU, runtime как ОС, навыки как приложения
